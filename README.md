# ContextApi_Store_Project

This project contains a React.js application for an e-commerce store. The application allows users to browse products, add items to their carts, and place orders.


## Technologies Used

- React.js
  
- React Router Dom
  
- Axios
  
- Context API
  
- Bootstrap
  
- useState Hook
